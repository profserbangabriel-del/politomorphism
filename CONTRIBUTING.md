# Contributing to Politomorphism

Thank you for your interest in contributing to the Politomorphism research project! Your help is valuable in advancing the study of political symbols and computational social science.

## How to contribute

### 1. Fork the repository
- Click the "Fork" button at the top right of the repository page.
- This creates a copy of the project in your GitHub account.

### 2. Create a branch
- In your fork, create a new branch for your changes:
  ```bash
  git checkout -b feature/your-feature-name
