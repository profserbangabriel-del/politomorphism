## What is Politomorphism?

Politomorphism is an original theoretical framework for analyzing 
the diffusion and transformation of political symbols. A symbol is 
politomorphic to the degree that the same signifying form condenses 
divergent meaning configurations for distinct political communities.

---

## Models

### Symbolic Resonance Mapping (SRM)
R(S, G) = A(G) · I(S, G) · C(G)

R_total(S) = Σ R(S, G_i)

- A(G) ∈ [0,1] — Affiliation factor  
- I(S,G) ∈ [−1,1] — Interpretation polarity  
- C(G) ∈ [0,1] — Cultural centrality  

### Entropic Equilibrium Function (EEF)
S(t) = −Σ p_i(t) · log(p_i(t))

dS/dt = Π(t) − Φ(t)
